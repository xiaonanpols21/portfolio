## Installeren van de portfolio

Zorg ervoor dat je het volgende op je computer heb geïnstalleerd: [Git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git), [Node](https://nodejs.org/en/download/) en [NPM](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm). 

``` bash

git clone https://github.com/xiaonanpols21/portfolio.git

```
``` bash

npm install

```

``` bash

npm run dev

```
